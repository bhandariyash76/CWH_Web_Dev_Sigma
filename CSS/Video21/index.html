CSS Specificity

Specificity is a set of rules that browsers use to determine which CSS rule applies when there are multiple rules that could apply to the same element. It is based on the types of selectors used in the CSS rules.

Specificity Hierarchy

  • Inline styles (e.g., style="color: red;") have the highest specificity.
  • ID selectors (e.g., #my-id) have a higher specificity than class selectors.
  • Class selectors (e.g., .my-class) have a higher specificity than element selectors.
  • Element selectors (e.g., div, p) have the lowest specificity.

Calculating Specificity

Specificity is calculated based on the number of each type of selector in a rule. The formula is:

    Specificity = (a, b, c, d)
    where:
    a = 1 if inline styles are present, otherwise 0
    b = number of ID selectors
    c = number of class selectors, attributes selectors, and pseudo-classes
    d = number of element selectors and pseudo-elements

Example

Consider the following CSS rules:

    /* Rule 1 */
    p {
        color: blue;
    }

    /* Rule 2 */
    .my-class {
        color: green;
    }

    /* Rule 3 */
    #my-id {
        color: red;
    }

If we have an HTML element like this:

    
Hello World!

The color of the text will be red because the ID selector (#my-id) has a higher specificity than the class selector (.my-class) and the element selector (p).
