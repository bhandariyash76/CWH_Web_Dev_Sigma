Fonts And Colors

This is a simple example of how to use CSS to set fonts and colors for a webpage. The body of the page uses the Arial font, has a text color of #333 (a dark gray), and a background color of #f0f0f0 (a light gray). Font properties allow you to specify the font family, size, weight, and style for text on your webpage. Color properties let you set the color of text and other elements using various formats such as hex codes, RGB, or named colors.
